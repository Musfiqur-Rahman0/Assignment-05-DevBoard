Question-5: How do you create, add, and remove elements using JavaScript?

  • To Create : 
    •        
        const div = document.createElement("div"); 
        <!-- this will create a new div -->
  • To Add : 
    •        
        const containerElement  = document.getElementById("container"); 
        <!-- this will get the container by id from the html file -->
        containerElement.appendChild(div); 
        <! this line will add the div that has created by js into the container.
  • To Remove : 
    •                         
    <!-- We can use removeChild() ,  remove() , element.innerHTML = ""  -->
        
  • removeChild() : This method removes a child element from the parent element
             container.removeChild(div)  
             <!-- This will remove the div from the container -->
        
    • 
        
  • remove() : This method removes a child element from the parent element
             container.remove()  
             <!-- This will remove the container itself. -->
        
    • 
        
  • element.innerHTML = "" : This method removes every  element from the parent element
