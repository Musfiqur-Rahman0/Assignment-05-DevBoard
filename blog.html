Question-1: What are the different ways to select an element in the DOM?

<!-- There are a lot of ways to select an html element using JavaScript in this blog i will talk about some most frequently used method to select a html element using JavaScript. -->

  • document.getElementById('idName') : In this method we just have to pass the id name of the element what we want to access. Example
    • const element = document.getElementById('btn')
  • document.getElementByClass('className') : In this method we just have to pass the class name of the element what we want to access. Example
    • const element = document.getElementByClassName('container')
  • document.getElementsByClassName('className') : In this method we just have to pass the class name of the element what we want to access. and this method will return all the element that has the same class name that just pass here. Example
    • const element = document.getElementById('box') This will return all the element that has the box class in a document list.
  • document.querySelector('id/class/tag name') : In this method we just have to pass the class/id/tag name of the element with the css selector for the id and the class what we want to access. and this method will return the first element that has the same class name that just pass here. Example
    • const element = document.querySelector('box') This will return the first element that has the box class.
  • document.querySelectorAll('id/class/tag name') : In this method we just have to pass the class/id/tag name with the css selector for id or class of the element what we want to access . and this method will return the all element that has the same class name that just pass here in a nodelist. Example
    • const element = document.querySelectorAll('box') This will return all the element that has the box class in a nodelist.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

  • innerHTML : innerHTML return all text inside an element including html tags and comments
  • innerText : innerText only return all the visible text inside an element including it's child element text .But ingnores the hidden text like if you give a element display none and if you try to access the innerText the output will be empty.
  • textContent : gets or sets all the text content even the hidden one. but not returns any html tags
